# broccoli-testem

> [broccoli](https://github.com/broccolijs/broccoli) plugin for running testem

## Usage

First, add `broccoli-testem` as a dependency:

```shell
npm install -g broccoli-testem
```

Then, run:

```shell
broccoli-testem
```
